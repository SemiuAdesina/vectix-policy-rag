# SOP: Expired Drug Inventory Management

**Policy ID:** VL-OPS-003  
**Effective Date:** January 1, 2026  
**Focus:** Weekly audits and Schedule II drug handling.  
**Version:** 2.0.1

## 1. Weekly Audits

Pharmacists at UCH utilizing VectixLogic software must perform a physical count of "Schedule II" controlled substances every Friday by 4:00 PM.

## 2. Disposal Procedure

Drugs expiring within 30 days must be flagged in the system. Expired items are moved to the "Quarantine Zone" and must be witnessed by two licensed pharmacists during destruction.

## 3. Revenue Recovery

The system automatically generates a "Return for Credit" (RFC) form for eligible manufacturers to recover costs on unsold inventory.

## 5. Appendix: Definitions

| Term | Definition |
|------|------------|
| **Schedule II** | US DEA schedule for controlled substances with high abuse potential; requires strict inventory controls. |
| **Weekly audit** | Physical count of Schedule II (and per-site other products) every Friday by 4:00 PM. |
| **Quarantine Zone** | System state for expired or suspect product; no dispense until cleared or destroyed. |
| **RFC** | Return for Credit; process to return eligible expired/short-dated product to manufacturer for credit. |
| **UCH** | University College Hospital, Ibadan. |
| **Licensed pharmacist** | Individual holding current PCN (or equivalent) license; required for witness. |
| **Destruction** | Lawful disposal of expired or illegitimate product; must be witnessed where required. |
| **30 days** | Window within expiry by which product must be flagged in the system. |
| **Eligible manufacturers** | Manufacturers with active RFC program per VectixLogic eligibility list. |
| **Physical count** | Actual count of units on hand; must match system after reconciliation. |
| **Flagged** | Marked in the system (e.g., Expiring Soon report) for review or action. |
| **Revenue recovery** | Process (e.g., RFC) to recover value from unsold or expired inventory. |
| **SOP** | Standard Operating Procedure. |
| **PCN** | Pharmacists Council of Nigeria; see VL-COMP-011. |
| **VectixLogic software** | Pharmacy inventory system covered by this SOP. |
